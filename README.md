# number-guessing-game
🎮 Zahlen-Ratespiel im Browser – gebaut mit HTML, CSS &amp; JavaScript
## Spielprinzip
- Das Spiel wählt eine zufällige Zahl zwischen 1 und 100
- Du hast X Versuche, die Zahl zu erraten
- Nach jedem Versuch bekommst du einen Hinweis: zu hoch / zu niedrig


## Technologien
- HTML5
- CSS3
- Vanilla JavaScript (kein Framework)

## Installation
1. Repository klonen:
   git clone
2. index.html im Browser öffnen – fertig!
